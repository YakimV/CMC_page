CMC - Customizable Mouse Controller

Give your mouse more options with CMC!

About the Application

Customize the second mouse as an additional tool!

The program allows you to use a second mouse to launch shortcuts, files, or scripts. Assign any actions to its buttons and automate routine tasks. A simple interface and flexible settings - everything for your convenience!

The program is actively developing, I recommend checking sources for the release of a new version of CMC

CMC Setup Guide

1. Download the Application

Download the application from one of the sources provided to you. Make sure the application runs successfully on your computer.

2. Installing a Special Driver for the Mouse

Download the LibUSB driver. It is recommended to use the Zadig utility for installation.

  1. Open Zadig and follow these steps:
  2. Go to the Options menu → select List All Devices.
  3. Find your mouse in the list.
    Tip: Disconnect and reconnect the mouse to identify it in the list.
  4. Some mice may have multiple interfaces (e.g., for different buttons or functions). Typically, interface 0 corresponds to the main buttons, so select it.
  5. In the driver field, choose LibusbK or Libusb (both will work).
  6. Click the Install Driver button.

3. Obtaining Vendor ID and Product ID

After installing the driver, copy two values in USB ID:

  • Vendor ID (left parameter).
  • Product ID (right parameter).

Insert these values into the corresponding fields in the application. To unlock these fields, use the appropriate button in the application's interface.

4. Configuring Functions for Mouse Buttons

After setting up the driver, you can assign functions to the mouse buttons:

  • Select ready-made functions from the dropdown menu.
  • To create custom shortcuts:
    • Write the shortcut in the field in the format: key:ctrl+c
    • (Almost all keys are supported. You can view their list in the application's hints.)
  • To launch files or scripts:
    • Specify the file path in the corresponding field.
    • You can also use the button to select a file.

5. ALSO...

Audio files are played through your default audio player. Files in the .wav format are played in the background without opening the player.

Testing took place on 8-bit mice and windows 11

To return the mouse driver to the standard version, just search for something like libusb in the device manager and uninstall the device, if possible, be sure to check the “try to uninstall the driver” box, and then reconnect the mouse.
